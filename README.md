# LNJPIT Chatbot

A simple AI-based chatbot web application developed for providing information about Loknayak Jai Prakash Institute of Technology (LNJPIT), Chhapra.

## Features
- College information chatbot
- Simple and interactive chat interface
- Flask backend integration
- HTML, CSS, and Python based project
- User-friendly design

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Cohere API

## Installation

1. Clone the repository

git clone <your-repository-link>

2. Move into project folder

cd project-name

3. Install dependencies

pip install flask cohere

4. Run the application

python app.py

5. Open browser and visit

http://127.0.0.1:5000/
